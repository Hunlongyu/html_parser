HTML Tokenizer 测试页面

此页面包含各种HTML结构，用于测试HTML解析器的Tokenizer功能

基础标签测试

1. 简单开始和结束标签
<div>Hello World</div>
预期Tokens:
StartTag(div), Text(Hello World), EndTag(div)
2. 自闭合标签
<img src="image.jpg" alt="测试图片" />
预期Tokens:
StartTag(img) [属性: src="image.jpg", alt="测试图片"]
3. 嵌套标签
<div><p>段落<span>内容</span></p></div>
预期Tokens:
StartTag(div), StartTag(p), Text(段落), StartTag(span), Text(内容), EndTag(span), EndTag(p), EndTag(div)

属性测试

1. 带引号的属性
<input type="text" name="username" value="test" />
预期Tokens:
StartTag(input) [属性: type="text", name="username", value="test"]
2. 不带引号的属性
<div class=container id=main>内容</div>
预期Tokens:
StartTag(div) [属性: class=container, id=main], Text(内容), EndTag(div)
3. 布尔属性
<input disabled checked />
预期Tokens:
StartTag(input) [属性: disabled, checked]
4. 特殊字符属性值
<div data-value="<script>alert('test')</script>"></div>
预期Tokens:
StartTag(div) [属性: data-value="<script>alert('test')</script>"]

特殊内容测试

1. 注释
<!-- 这是注释内容 -->
预期Tokens:
Comment(这是注释内容)
2. DOCTYPE声明
<!DOCTYPE html>
预期Tokens:
DOCTYPE(html)
3. 字符实体
<p>HTML中的特殊字符: &lt;, &gt;, &amp;, &quot;</p>
预期Tokens:
StartTag(p), Text(HTML中的特殊字符: <, >, &, "), EndTag(p)
4. 原始文本元素（script和style）
<script>if (a < b && b > c) { alert('test'); }</script>
预期Tokens:
StartTag(script), Text(if (a < b && b > c) { alert('test'); }), EndTag(script)

复杂结构测试

1. 复杂嵌套结构
<div class="container"> <header> <h1>标题</h1> <nav> <ul> <li><a href="#">链接1</a></li> <li><a href="#">链接2</a></li> </ul> </nav> </header> <main> <article> <h2>文章标题</h2> <p>段落内容</p> </article> </main> </div>
预期Tokens:
约30个Tokens
2. 表单元素
<form action="/submit" method="post"> <label for="name">姓名:</label> <input type="text" id="name" name="name" required> <br> <label><input type="checkbox" name="agree"> 我同意条款</label> <br> <button type="submit">提交</button> </form>
预期Tokens:
约20个Tokens
测试说明

此页面设计用于测试HTML解析器的Tokenizer功能。每个测试用例都包含了HTML代码和期望的Token输出。

要使用此页面进行测试，可以将每个测试用例的HTML代码输入到您的Tokenizer中，然后检查输出的Token序列是否与预期结果匹配。

注意：实际测试时，Tokenizer应该能够处理整个HTML文档，而不仅仅是片段。
